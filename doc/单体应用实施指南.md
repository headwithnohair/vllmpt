- ✅ 新项目快速启动

##### 2. 输入类型识别器

**位置**：`ai/recognizer/InputTypeRecognizer.java`

**识别逻辑**：
```java
public InputType recognize(ChatRequest request) {
    boolean hasText = StringUtils.hasText(request.getContent());
    boolean hasImage = !CollectionUtils.isEmpty(request.getImageUrls());
    boolean hasFile = !CollectionUtils.isEmpty(request.getFileUrls());
    
    if (hasFile) return InputType.FILE;
    if (hasText && hasImage) return InputType.MULTIMODAL;
    if (hasImage) return InputType.IMAGE_ONLY;
    return InputType.TEXT_ONLY;
}
```

##### 3. 场景识别器（关键词规则）

**位置**：`ai/recognizer/SceneRecognizer.java`

**实现思路**：

##### 4. 路由策略（策略模式）

**接口定义**：
```java
public interface RoutingStrategy {
    ModelInfo route(InputType inputType, BizScene scene, RoutingContext context);
}
```

**优先级路由策略实现**：
```java
@Component
public class PriorityRoutingStrategy implements RoutingStrategy {
    
    @Autowired
    private ModelRegistry modelRegistry;
    
    @Override
    public ModelInfo route(InputType inputType, BizScene scene, RoutingContext context) {
        List<ModelInfo> candidates = modelRegistry.filterModels(inputType, scene);
        
        if (candidates.isEmpty()) {
            throw new IllegalStateException("没有可用的模型");
        }
        
        // 返回优先级最高的模型（priority 最小）
        return candidates.get(0);
    }
}
```

##### 5. 模型路由器（核心整合）

**位置**：`ai/router/DefaultModelRouter.java`

**路由流程**：
```
1. 识别输入类型 → InputTypeRecognizer
2. 识别业务场景 → SceneRecognizer
3. 构建路由上下文 → RoutingContext
4. 执行路由策略 → RoutingStrategy.route()
5. 返回选中的模型 → ModelInfo
```

**关键代码思路**：
```java
@Component
public class DefaultModelRouter implements ModelRouter {
    
    @Autowired
    private InputTypeRecognizer inputTypeRecognizer;
    
    @Autowired
    private SceneRecognizer sceneRecognizer;
    
    @Autowired
    private RoutingStrategy routingStrategy;
    
    @Override
    public ModelInfo route(ChatRequest request) {
        // 1. 识别输入类型
        InputType inputType = inputTypeRecognizer.recognize(request);
        
        // 2. 识别业务场景
        BizScene scene = sceneRecognizer.recognize(request.getContent());
        
        // 3. 构建路由上下文
        RoutingContext context = RoutingContext.builder()
            .userId(request.getUserId())
            .sessionId(request.getSessionId())
            .build();
        
        // 4. 执行路由策略
        ModelInfo selectedModel = routingStrategy.route(inputType, scene, context);
        
        log.info("路由选择模型: {} (场景: {}, 类型: {})", 
            selectedModel.getModelName(), scene, inputType);
        
        return selectedModel;
    }
}
```

**关键代码思路**：
```java
@Component
public class SiliconFlowAdapter implements ModelAdapter {
    
    @Autowired
    private WebClient webClient;
    
    @Override
    public ChatResponse chat(ChatRequest request) {
        ModelInfo model = request.getSelectedModel();
        
        // 构建请求体
        SiliconFlowRequest apiRequest = buildRequest(request);
        
        // 调用 API
        SiliconFlowResponse apiResponse = webClient.post()
            .uri("/chat/completions")
            .bodyValue(apiRequest)
            .retrieve()
            .bodyToMono(SiliconFlowResponse.class)
            .timeout(Duration.ofSeconds(30))
            .retryWhen(Retry.backoff(3, Duration.ofSeconds(1)))
            .block();
        
        // 转换响应
        return convertResponse(apiResponse, model);
    }
    
    @Override
    public Flux<ChatResponse> chatStream(ChatRequest request) {
        // 类似同步调用，但使用 bodyToFlux 解析 SSE
    }
}
```

```
session:info:{sessionId}         → SessionVO（TTL 30分钟）
```

**Cache-Aside 模式**：
```java
public SessionVO getSessionWithCache(String sessionId) {
    String key = "session:info:" + sessionId;
    
    // 1. 先查缓存
    SessionVO cached = redisTemplate.opsForValue().get(key);
    if (cached != null) {
        return cached;
    }
    
    // 2. 缓存未命中，查数据库
    SessionVO session = sessionRepository.getBySessionId(sessionId);
    
    // 3. 写入缓存
    if (session != null) {
        redisTemplate.opsForValue().set(key, session, 30, TimeUnit.MINUTES);
    }
    
    return session;
}
```

##### 4. 上下文管理

**位置**：`chat/service/SessionService.java`

**实现思路**：
```java
public List<Message> getConversationContext(String sessionId, int maxMessages) {
    // 1. 先查 Redis 缓存
    String key = "session:messages:" + sessionId;
    List<Message> cached = redisTemplate.opsForList().range(key, -maxMessages, -1);
    
    if (!CollectionUtils.isEmpty(cached)) {
        return cached;
    }
    
    // 2. 缓存未命中，查数据库
    List<Message> messages = messageRepository.selectRecentBySessionId(sessionId, maxMessages);
    
    // 3. 写入缓存
    if (!CollectionUtils.isEmpty(messages)) {
        redisTemplate.opsForList().rightPushAll(key, messages);
        redisTemplate.expire(key, 10, TimeUnit.MINUTES);
    }
    
    return messages;
}
```

```java
@Component
@Slf4j
public class AiCallAspect {
    
    @Around("@annotation(org.albedo.vllmpt.common.annotation.AiCallLog)")
    public Object recordAiCall(ProceedingJoinPoint joinPoint) throws Throwable {
        long startTime = System.currentTimeMillis();
        
        try {
            // 执行目标方法
            Object result = joinPoint.proceed();
            
            long responseTime = System.currentTimeMillis() - startTime;
            
            // 记录成功日志
            logAiCall(joinPoint, result, responseTime, true);
            
            return result;
            
        } catch (Exception e) {
            long responseTime = System.currentTimeMillis() - startTime;
            
            // 记录失败日志
            logAiCall(joinPoint, null, responseTime, false);
            
            throw e;
        }
    }
}
```

```java
@Service
public class UsageStatsService {
    
    public void recordUsage(Long userId, int tokensUsed, BigDecimal cost) {
        String date = LocalDate.now().toString();
        
        // 调用次数
        String callsKey = String.format("stats:user:%s:%s:calls", userId, date);
        redisTemplate.opsForValue().increment(callsKey);
        
        // Token 消耗
        String tokensKey = String.format("stats:user:%s:%s:tokens", userId, date);
        redisTemplate.opsForValue().increment(tokensKey, tokensUsed);
        
        // 费用
        String costKey = String.format("stats:user:%s:%s:cost", userId, date);
        redisTemplate.opsForValue().increment(costKey, cost.doubleValue());
        
        // 设置过期时间（保留 30 天）
        redisTemplate.expire(callsKey, 30, TimeUnit.DAYS);
    }
}
```
